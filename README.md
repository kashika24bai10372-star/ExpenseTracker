Student Expense Tracker (Java)

Overview

This is a simple Java-based Expense Tracker application designed for students to manage their daily income and expenses efficiently.

The application allows users to record transactions, categorize expenses, and track their financial balance.

Features

* Add Income
* Add Expense
* View All Transactions
* View Current Balance
* Filter Transactions by Category
* Data Persistence using File Handling

Technologies Used

* Java (JDK)
* Object-Oriented Programming (OOP)
* File Handling
* VS Code

How to Run

1. Open terminal in project folder

2. Navigate to src folder:
   cd src

3. Compile:
   javac *.java

4. Run:
   java Main
